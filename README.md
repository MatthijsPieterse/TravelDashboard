# ✈️ My Personal Travel Log
Welcome to my personal travel log! This is where I keep track of all the countries and cities I’ve explored. Each entry can include different types of travel tips, like:
🍴 Food & Drinks – restaurants, cafes, bars, and local specialties
🏨 Accommodations – hotels, hostels, or unique stays
🎡 Activities & Attractions – sightseeing, tours, adventures, and local experiences
💡 General Travel Tips – best times to visit, transport advice, or practical recommendations

### 🌍 Features
- Travel Dashboard
- Travel Tips – Recommendations for:
    - 🍴 Food & Drinks
    - 🏨 Accommodation
    - 🎡 Activities & attractions
    - 💡 General Travel Tips
- Static JSON Data – All entries are stored in simple JSON and GeoJSON files

### 🛠️ Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
